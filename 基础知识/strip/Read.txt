strip：英文是脱去,旅行的意思

函数原型
声明：s为字符串，cc为要删除的字符序列
s.strip(cc)        删除s字符串中开头、结尾处，位于 cc删除序列的字符
s.lstrip(cc)       删除s字符串中开头处，位于 cc删除序列的字符
s.rstrip(cc)      删除s字符串中结尾处，位于 cc删除序列的字符

注意：

(1)当cc为空时，默认删除空白符（包括'\n', '\r',  '\t',  ' ')

>>> a = '     123'
>>> a.strip()
'123'
>>> a='\t\tabc'
'abc'
>>> a = 'sdff\r\n'
>>> a.strip()
'sdff'

(2)这里的cc删除序列是只要边（开头或结尾）上的字符在删除序列内，就删除掉。

>>> a = '123abc'
>>> a.strip('21')
'3abc'   结果是一样的
>>> a.strip('12')
'3abc'
